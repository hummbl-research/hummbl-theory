## Core Thesis

HUMMBL is a **governed mental-model layer** (Base120) **for reasoning over partial world models**, with append-only audit trails as proof of sovereignty (VERUM), creating the belonging infrastructure (BKI) that makes AI governance adoption possible.

These three claims map onto three theoretical traditions:
- **Base120** ← Kleinian transformation geometry + non-commutative algebra
- **HUMMBL governance** ← Ashby's Law of Requisite Variety + Turchin Metasystem Transition Theory
- **BKI** ← Scale-free network dynamics (Barabási) + Kantian categorical pluralism + Ubuntu relational ontology
