## Thesis

Multi-agent AI systems require governance infrastructure by mathematical necessity, not merely as a best practice. This follows directly from Ashby's Law of Requisite Variety (1956) and the Conant-Ashby Good Regulator Theorem (1970): a regulator can control a system only if it has variety ≥ the governed system's disturbance variety, and any optimal regulator must contain an isomorphic model of the regulated system.

## Core Claims

1. **Governance is mathematically necessary** — Ashby's Law applied to multi-agent systems
2. **Governance requires world modeling** — Conant-Ashby applied to the governance layer
3. **Base120 provides the governance mental-model layer** — Klein's Erlangen Programme applied to cognitive transformation operators
4. **BKI infrastructure prevents epistemic hegemony** — Barabási scale-free networks + Kantian categorical pluralism
