# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository or any HUMMBL project, please report it responsibly.

**Email**: reuben@hummbl.io

Please include:
- Description of the issue
- Steps to reproduce (if applicable)
- Any relevant file paths or references

## Response Timeline

We aim to acknowledge reports within 48 hours and provide a resolution timeline within 7 days.

## Scope

This is a documentation/theory repository. Security concerns here are most likely:
- Accidentally committed secrets or credentials
- Sensitive information in theoretical documents
- Supply chain issues in any tooling dependencies
